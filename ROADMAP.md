# Roadmap

## 0.1 MVP

- Local markdown discovery.
- Local link and file-reference checks.
- Package script command checks.
- README install and usage coverage hints.
- Explicit `docfresh: smoke` command execution.
- Text and JSON reports.
- Fixture-backed tests.

## Next

- Validate markdown heading fragments for local links.
- Add config files for ignores and severity overrides.
- Add SARIF or GitHub annotations output.
- Add richer command parsing for multiline shell examples.
- Publish npm package after release automation is reviewed.

## Non-goals

- Grammar or prose style linting.
- Hosted link checking by default.
- Running arbitrary command blocks without explicit opt-in.
